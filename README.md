# 🍵 Tea Shop ChatBot

> An AI-powered chatbot that helps customers explore the menu, place orders, and get answers — all in a friendly, conversational way.

---

## 📌 Project Overview

The **Tea Shop ChatBot** is a conversational assistant built for a tea shop. Customers can interact with the bot to:

- Browse the tea menu
- Ask about ingredients and prices
- Place or customize orders
- Get recommendations based on their mood or preference
- Get information about shop timings and location

---

## 🚀 Features

- 💬 Natural language conversation
- 🍵 Full tea menu with descriptions
- 🛒 Order placement via chat
- 📍 Shop info (location, hours)
- 🤖 AI-powered responses
- 📱 Simple and clean UI

## ⚙️ How to Run

### 1. Clone the Repository

```bash
git clone https://github.com/seeksforai-alt/Tea-Shop-Chatbot.git
cd Tea-Shop-Chatbot
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Run the App

```bash
python app.py
```

### 4. Open in Browser

```
http://localhost:5000
```

## 💬 Sample Conversation

```
User:   Hi! What teas do you have?
Bot:    Hello! 🍵 We have Green Tea, Masala Chai, Kashmiri Pink Tea,
        Mint Tea, and more! What sounds good to you?

Bot:    Great choice! One Kashmiri Pink Tea added to your order. ☕
        Anything else?
